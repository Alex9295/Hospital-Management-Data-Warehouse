##🏥 Hospital Operations Analytics (SQL Project)
#📌 Project Overview

This project simulates a relational hospital management system and applies advanced SQL queries to analyze operational performance, financial metrics, and patient trends.

The goal was to demonstrate real-world data analytics using SQL — including data cleansing, joins across multiple tables, KPI calculations, and performance optimization.

This project focuses on answering business-critical questions such as:

Which departments generate the highest revenue?

What is the average patient length of stay?

Which diagnoses have the highest readmission rates?

How do admission trends vary over time?

Which physicians or service lines are most utilized?
